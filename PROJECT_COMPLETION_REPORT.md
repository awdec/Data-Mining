## 🏗️ 系统架构实现

### 数据流架构
```
Codeforces API → CSV存储 → 数据预处理 → 机器学习/Web分析 → 用户界面
```

### 核心组件
1. **数据采集层**: `api.py` - 带重试机制的API数据获取
2. **数据处理层**: 数据清洗、特征工程、标签处理
3. **分析层**: 机器学习建模 + 统计分析
4. **应用层**: Flask Web服务 + 缓存机制
5. **表现层**: 响应式Web界面 + 动态图表
